# Calculadora com Classes

## 📖 Descrição

Projeto desenvolvido em Java para praticar os fundamentos da Programação Orientada a Objetos (POO). O programa recebe dois números inteiros informados pelo usuário, cria um objeto da classe e calcula o resultado utilizando um método.

## 🚀 Funcionalidades

- Recebe dois números inteiros informados pelo usuário.
- Permite escolher a operação matemática:
  - Soma (`+`)
  - Subtração (`-`)
  - Multiplicação (`*`)
  - Divisão (`/`)
- Valida a divisão por zero.
- Exibe uma mensagem para operadores inválidos.
- Exibe o resultado no console.

## 💻 Exemplo

```text
=== Calculadora ===

Digite o primeiro número: 20
Digite um operador (+, -, *, /):
*

Digite o segundo número: 5

A multiplicação de 20 * 5 = 100
```

### Exemplo de validação

```text
Digite o primeiro número: 10
Digite um operador (+, -, *, /):
/

Digite o segundo número: 0

Não é possível dividir por zero!
Por favor, escolha outro número:

2

A divisão de 10 / 2 = 5
```

## 🎯 Objetivo

Praticar os fundamentos da Programação Orientada a Objetos em Java por meio de um projeto que utiliza classes, objetos, construtores, métodos e validação de dados para realizar operações matemáticas básicas.
